The Assignment¶

Take a ZIP file) of images and process them, using a library built into python that you need to learn how to use. A ZIP file takes several different files and compresses them, thus saving space, into one single file. The files in the ZIP file we provide are newspaper images (like you saw in week 3). Your task is to write python code which allows one to search through the images looking for the occurrences of keywords and faces. E.g. if you search for "pizza" it will return a contact sheet of all of the faces which were located on the newspaper page which mentions "pizza". This will test your ability to learn a new (library), your ability to use OpenCV to detect faces, your ability to use tesseract to do optical character recognition, and your ability to use PIL to composite images together into contact sheets.

Each page of the newspapers is saved as a single PNG image in a file called images.zip. These newspapers are in english, and contain a variety of stories, advertisements and images. Note: This file is fairly large (~200 MB) and may take some time to work with, I would encourage you to use small_img.zip for testing.

import zipfile
from PIL import Image
import pytesseract
import cv2 as cv
import numpy as np
from IPython.display import display

# loading the face detection classifier
face_cascade = cv.CascadeClassifier('readonly/haarcascade_frontalface_default.xml')

# Function to return the text recognised from the image
def get_text(image):
    '''Takes in an PIL.Image object and return the text contained in the image
    :param image: PIL.Image object
    :returns: string object containing the text in the image
    '''
    img = image.copy()
    img.convert('L')
    return pytesseract.image_to_string(img).replace('-\n', '')

# Function to return the contact sheet of faces in that image else a string if there are no faces
def get_faces(image):
    '''Takes in an PIL.Image object and returns the contact sheet
        of the faces contained in the image if there are faces else returns
        a string indicating no images are there.
    :param image: PIL.Image object
    :returns: PIL.Image object of the contact sheet or a string
    '''
    try:
        
        bounding_box = face_cascade.detectMultiScale(np.array(image),1.30,4).tolist()
        faces = []
        for x,y,w,h in bounding_box:
            faces.append(image.crop((x, y, x+w, y+h)))
            
        contact_sheet = Image.new(image.mode, (600, 120*int(np.ceil(len(faces)/5))))
    
        x = 0
        y = 0
            
        for face in faces:
            face.thumbnail((120,120))
            contact_sheet.paste(face,(x, y))
            if x+120 == contact_sheet.width:
                x = 0
                y = y + 120
            else:
                x = x + 120
                
        return contact_sheet
    except:
        return "But there were no faces in that file!"

# the rest is up to you!
# Opening a ZIP file
myzip = zipfile.ZipFile("readonly/images.zip")  # Main dataset
# myzip = zipfile.ZipFile("readonly/small_img.zip")  # Testing dataset

# Storing the filenames and images in a list of dictionaries
file_names = [x.filename for x in myzip.infolist()]
images = [{x: Image.open(myzip.open(x))} for x in file_names]
# Adding the text and contact sheet to the dictionaries of each image
for i,x in enumerate(images):
        x.setdefault('text',get_text(x[file_names[i]]))
        x.setdefault('faces', get_faces(x[file_names[i]]))

# Function that searches the text and displays the contact sheet if text is found
# Note that the processing needs to be done only once and the results can be obtained
# faster as now only a word is searched in the text
def search(text):
    '''Takes in the text to be searched and prints the filename the text was
        found in and displays a contact sheet with faces in that file.
    :param text: string containing word to be searched
    '''
    for index,img in enumerate(images):
        if text in img['text']:
            print("Results found in file", file_names[index])
            display(img['faces'])
